## Machine Learning Datasets
*Datasets widely used for training, benchmarking, and evaluating machine learning models across domains like computer vision, NLP, speech recognition, and recommendation systems.*

### A

| **Dataset** | **Description** | **Source / Documentation** |
|-------------|-----------------|----------------------------|
| Abalone Dataset | Dataset used for predicting the age of abalone from physical measurements; commonly used for regression problems | [Check Out](https://archive.ics.uci.edu/ml/datasets/abalone) |
| Adult Census Income Dataset | Dataset used for predicting whether income exceeds $50K/year based on census data; widely used for classification tasks | [Check Out](https://archive.ics.uci.edu/ml/datasets/adult) |
| Amazon Reviews Dataset | Large-scale dataset containing product reviews used for sentiment analysis, recommendation systems, and NLP research | [Check Out](https://nijianmo.github.io/amazon/index.html) |
| Amazon Product Data | Dataset containing product metadata, reviews, and ratings for building recommendation systems | [Check Out](https://jmcauley.ucsd.edu/data/amazon/) |
| Animals with Attributes (AwA) Dataset | Image dataset annotated with semantic attributes used in computer vision and zero-shot learning research | [Check Out](https://cvml.ista.ac.at/AwA/) |
| AudioSet | Large-scale dataset of labeled audio events from YouTube used for audio classification and sound recognition | [Check Out](https://research.google.com/audioset/) |
| AVA Action Dataset | Video dataset for spatio-temporal action detection in movies | [Check Out](https://research.google.com/ava/) |
| Arabic Speech Corpus | Speech dataset for Arabic automatic speech recognition research | [Check Out](https://arabicspeech.org/) |
| ArXiv Dataset | Large collection of research paper metadata used for NLP, recommendation systems, and knowledge graph tasks | [Check Out](https://www.kaggle.com/Cornell-University/arxiv) |
| Affective Text Dataset | Dataset used for emotion recognition and sentiment analysis in NLP | [Check Out](https://www.kaggle.com/datasets) |
